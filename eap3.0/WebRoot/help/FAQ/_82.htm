6.7.客商、合同为何审核通过后，状态不改变呢？

 

客商、合同为何审核通过后，状态不改变呢？

 

需要从系统管理――流程设置，找到新建客商档案，打开改流程，点击最后一个节点，右边的关联执行，复制这段语句后保存

update cs_customer set istatus=1 where iid=@iid@ and '@wftype@'='处理' update cs_customer set istatus=0 where iid=@iid@ and '@wftype@'='撤销'

每个合同的操作也是同样，只是复制的语句不一样，合同的语句如下

update sc_order set istatus=1 where iid=@iid@ and '@wftype@'='处理' update sc_order set istatus=0 where iid=@iid@ and '@wftype@'='撤销'
