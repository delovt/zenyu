7.1.3 服务工单

【功能概述】

    服务部负责人填写工单并将工单派到某一服务人员那，服务人员进行服务之后，要填写服务记录。该工单记录了服务类型、服务人员、问题描述、处理过程以及解决情况。便于服务部负责人，直观地了解服务的情况，更好地进行下一步的服务工作。

2、专用区

列表1：围绕服务工单展开的服务工单管理查询配置。

列表2：更多操作

        分配人员：对服务工单批量分配服务人员，在弹出对话框中选择人员，点击确定即可。

单据1：更多操作

        服务记录：一般由服务人员从客户处服务回来后填写服务记录，在弹出的对话框中输入相关信息即可，点击确定则工单的状态更新为填单。

        服务交单：服务交单说明这次服务事务的结束，用于记录工单的状态，提交工单后，服务工单的状态更新为关闭，如需更改工单，则需要点击工单打开。

        生成回访：由服务工单生成服务回访，进行后续的回访记录跟踪。服务工单状态为关闭的才可以生成回访。

        撤销记录：填写服务记录后，可以通过撤销记录清除单据中的服务记录信息。

        撤销交单：服务交单后，如果不是全部解决，可以点击撤销交单来进行服务记录的填写，服务工单的状态更新为填单。

        工单关闭：更新服务工单的状态为关闭，关闭的工单是无法进行修改、删除等其他的单据操作。需要工单打开后在进行操作。

        工单打开：打开服务工单。

        再派服务：对于处理结果不是全部解决的服务工单，可以再次进行派单的操作，系统会自动生成一张新的服务工单。

 

服务工单的状态为：派单、填单、交单、关闭

逆向操作服务工单的顺序为：

工单打开 -> 撤销交单 -> 撤销记录
